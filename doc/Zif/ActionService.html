Overview

This just facilitates keeping track of and running Actionables that need updating every tick. Specifically, when #run_all_actions is invoked this will call #perform_actions on all Actionable objects which have been previously registered using #register_actionable.

def run_all_actions
  actionables_count = @actionables&.length

  return false unless actionables_count&.positive?

  # Avoid blocks here.
  idx = 0
  while idx < actionables_count
    @actionables[idx].perform_actions
    idx += 1
  end

  true
end
